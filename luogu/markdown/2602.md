# [ZJOI2010]数字计数

## 题目描述

给定两个正整数a和b，求在[a,b]中的所有整数中，每个数码(digit)各出现了多少次。

## 输入输出格式

### 输入格式

输入文件中仅包含一行两个整数a、b，含义如上所述。

### 输出格式

输出文件中包含一行10个整数，分别表示0-9在[a,b]中出现了多少次。

## 输入输出样例

### 输入样例 #1

```cpp
1 99

```
### 输出样例 #1

```cpp
9 20 20 20 20 20 20 20 20 20

```
## 说明

30%的数据中，a<=b<=10^6；

100%的数据中，a<=b<=10^12。
