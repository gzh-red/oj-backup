# 对角线

## 题目描述

对于一个N个定点的凸多边形，他的任何三条对角线都不会交于一点。请求楚图形中对角线交点的个数。

例如，6边形：

![](https://cdn.luogu.com.cn/upload/pic/6023.png)

## 输入输出格式

### 输入格式

第一行一个n，代表边数。

### 输出格式

第一行输出交点数量

## 输入输出样例

### 输入样例 #1

```cpp
3
```


### 输出样例 #1

```cpp
0
```


### 输入样例 #2

```cpp
6
```


### 输出样例 #2

```cpp
15
```


## 说明

50%的测试数据 3≤N≤100;

100%的测试数据 3≤N≤100000.
