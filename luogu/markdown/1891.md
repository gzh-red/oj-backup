# 疯狂LCM

## 题目描述

描述：

众所周知，czmppppp是数学大神犇。一天，他给众蒟蒻们出了一道数论题，蒟蒻们都惊呆了。。。

给定正整数N，求LCM(1,N)+LCM(2,N)+...+LCM(N,N)。

## 输入输出格式

### 输入格式

第一行一个数T，表示有T组数据。

对于每组数据，一行，一个正整数N。

### 输出格式

T行，每行为对应答案。

## 输入输出样例

### 输入样例 #1

```cpp
3
1
2
5
```


### 输出样例 #1

```cpp
1
4
55
```


## 说明

对于30%的数据，1≤T≤5，1≤N≤100000

对于100%的数据，1≤T≤300000，1≤N≤1000000
