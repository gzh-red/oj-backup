# 【模板】卢卡斯定理

## 题目背景

这是一道模板题。

## 题目描述

给定n,m,p($1\le n,m,p\le 10^5$)

求 $C_{n+m}^{m}\ mod\ p$

保证P为prime

C表示组合数。

一个测试点内包含多组数据。

## 输入输出格式

### 输入格式

第一行一个整数T($T\le 10$)，表示数据组数

第二行开始共T行，每行三个数n m p，意义如上

### 输出格式

共T行，每行一个整数表示答案。

## 输入输出样例

### 输入样例 #1

```cpp
2
1 2 5
2 1 5
```


### 输出样例 #1

```cpp
3
3
```
