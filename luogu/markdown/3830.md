# [SHOI2012]随机树

## 题目背景

SHOI2012 D1T3

## 题目描述

![](https://cdn.luogu.com.cn/upload/pic/6555.png)

## 输入输出格式

### 输入格式

输入仅有一行，包含两个正整数 q, n，分别表示问题编号以及叶结点的个数。

### 输出格式

输出仅有一行，包含一个实数 d，四舍五入精确到小数点后 6 位。如果 q = 1，则 d 表示叶结点平均深度的数学期望值；如果 q = 2，则 d 表示树深度的数学期望值。

## 输入输出样例

### 输入样例 #1

```cpp
1 4
```


### 输出样例 #1

```cpp
2.166667
```


### 输入样例 #2

```cpp
2 4
```


### 输出样例 #2

```cpp
2.666667
```


### 输入样例 #3

```cpp
1 12
```


### 输出样例 #3

```cpp
4.206421
```


### 输入样例 #4

```cpp
2 12
```


### 输出样例 #4

```cpp
5.916614
```
