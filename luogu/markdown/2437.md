# 蜜蜂路线

## 题目背景

无

## 题目描述

一只蜜蜂在下图所示的数字蜂房上爬动,已知它只能从标号小的蜂房爬到标号大的相邻蜂房,现在问你：蜜蜂从蜂房M开始爬到蜂房N，M<N，有多少种爬行路线？ ![](https://cdn.luogu.com.cn/upload/pic/1575.png) 

## 输入输出格式

### 输入格式

输入M,N的值

### 输出格式

爬行有多少种路线

## 输入输出样例

### 输入样例 #1

```cpp
1 14
```


### 输出样例 #1

```cpp
377
```


## 说明

对于100%的数据，$M,N\le 1000$
