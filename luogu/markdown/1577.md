# 切绳子

## 题目描述

有N条绳子，它们的长度分别为Li。如果从它们中切割出K条长度相同的

绳子，这K条绳子每条最长能有多长？答案保留到小数点后2位(直接舍掉2为后的小数)。

## 输入输出格式

### 输入格式

第一行两个整数N和K，接下来N行，描述了每条绳子的长度Li。

### 输出格式

切割后每条绳子的最大长度。

## 输入输出样例

### 输入样例 #1

```cpp
4 11
8.02
7.43
4.57
5.39
```


### 输出样例 #1

```cpp
2.00
```


## 说明

对于100%的数据 0<Li<=100000.00 0<n<=10000 0<k<=10000
