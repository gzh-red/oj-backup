# 【模板】自适应辛普森法2

## 题目描述

试计算积分

$\displaystyle{\int_0^\infty x^{\frac{a}{x}-x}\mathrm{d}x}$

保留至小数点后$5$位。若积分发散，请输出$\text{orz}$。

## 输入输出格式

### 输入格式

一行，包含一个实数，为$a$的值

### 输出格式

一行，积分值或$\text{orz}$

## 输入输出样例

### 输入样例 #1

```cpp
2.33
```


### 输出样例 #1

```cpp
1.51068
```


## 说明

$a\le50$

请注意时空限制。
