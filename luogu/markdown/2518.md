# [HAOI2010]计数

## 题目描述

你有一组非零数字（不一定唯一），你可以在其中插入任意个0，这样就可以产生无限个数。比如说给定{1,2},那么可以生成数字12,21,102,120,201,210,1002,1020,等等。

现在给定一个数，问在这个数之前有多少个数。（注意这个数不会有前导0）.

## 输入输出格式

### 输入格式

只有1行，为1个整数n.

### 输出格式

只有整数，表示N之前出现的数的个数。

## 输入输出样例

### 输入样例 #1

```cpp
1020
```


### 输出样例 #1

```cpp
7
```


## 说明

n的长度不超过50，答案不超过2^63-1.
