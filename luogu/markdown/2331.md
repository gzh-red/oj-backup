# [SCOI2005]最大子矩阵

## 题目描述

这里有一个n\*m的矩阵，请你选出其中k个子矩阵，使得这个k个子矩阵分值之和最大。注意：选出的k个子矩阵不能相互重叠。

## 输入输出格式

### 输入格式

第一行为n,m,k（1≤n≤100,1≤m≤2,1≤k≤10），接下来n行描述矩阵每行中的每个元素的分值(每个元素的分值的绝对值不超过32767)。

### 输出格式

只有一行为k个子矩阵分值之和最大为多少。

## 输入输出样例

### 输入样例 #1

```cpp
3 2 2
1 -3
2 3
-2 3

```
### 输出样例 #1

```cpp
9
```
