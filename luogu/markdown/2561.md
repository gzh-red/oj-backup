## 输入输出格式

### 输入格式

文件中以一行的形式存放一个正整数 n ， n ≤ 20 。

### 输出格式

以一行的形式输出问题的解 s (解的位数不超过 200 位)。

## 输入输出样例

### 输入样例 #1

```cpp
1
```


### 输出样例 #1

```cpp
2
```


### 输入样例 #2

```cpp
2
```


### 输出样例 #2

```cpp
4
```
