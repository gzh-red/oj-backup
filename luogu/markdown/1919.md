# 【模板】A*B Problem升级版（FFT快速傅里叶）

## 题目描述

给出两个n位10进制整数x和y，你需要计算x\*y。

## 输入输出格式

### 输入格式

第一行一个正整数n。 第二行描述一个位数为n的正整数x。 第三行描述一个位数为n的正整数y。

### 输出格式

输出一行，即x\*y的结果。（注意判断前导0）

## 输入输出样例

### 输入样例 #1

```cpp
1
3
4
```


### 输出样例 #1

```cpp
12
```
