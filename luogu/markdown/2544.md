# [AHOI2004]数字迷阵

## 题目描述

![](https://cdn.luogu.com.cn/upload/pic/1655.png)

## 输入输出格式

### 输入格式

每行有三个正整数，分别是i,j,m，其中i,j<=10^9,2<=m

### 输出格式

每行输出对应的第i行，第j列的那个正整数对m取模的结果。

## 输入输出样例

### 输入样例 #1

```cpp
1 2 99
```


### 输出样例 #1

```cpp
2
```


### 输入样例 #2

```cpp
9 1 999
```


### 输出样例 #2

```cpp
22
```
