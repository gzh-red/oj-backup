# A+B Problem（再升级）

## 题目背景

·题目名称是吸引你点进来的

·实际上该题还是很水的

## 题目描述

·1+1=? 显然是2

·a+b=? 1001回看不谢

·哥德巴赫猜想 似乎已呈泛滥趋势

·以上纯属个人吐槽

·给定一个正整数n，求将其分解成若干个素数之和的方案总数。

## 输入输出格式

### 输入格式

一行：一个正整数n

### 输出格式

一行：一个整数表示方案总数

## 输入输出样例

### 输入样例 #1

```cpp
7
```


### 输出样例 #1

```cpp
3
```


## 说明

【样例解释】

7=7

7=2+5

7=2+2+3

【福利数据】

【输入】 20

【输出】 26

【数据范围及约定】

对于30%的数据 1<=n

对于100%的数据，1<=n
