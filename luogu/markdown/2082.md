# 区间覆盖(加强版)

## 题目描述

已知有N个区间，每个区间的范围是[si,ti]，请求出区间覆盖后的总长。

## 输入输出格式

### 输入格式

N

s1 t1

s2 t2

……

sn tn

### 输出格式

共一行，一个正整数，为覆盖后的区间总长。

## 输入输出样例

### 输入样例 #1

```cpp
3
1 100000
200001 1000000
100000000 100000001
```


### 输出样例 #1

```cpp
900002
```


## 说明

【数据范围】

对于40%的数据 N≤1000，0<Si<Ti≤10000 对于100%的数据 N≤10^5，0<Si<Ti≤10^17，且为整数
