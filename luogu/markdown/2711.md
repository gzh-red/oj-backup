# 小行星

## 题目描述

星云中有n颗行星，每颗行星的位置是(x,y,z)。每次可以消除一个面（即x,y或z坐标相等）的行星，但是由于时间有限，求消除这些行星的最少次数。

## 输入输出格式

### 输入格式

第1行为小行星个数n，第2行至第n+1行为xi, yi, zi，描述第i个小行星所在的位置。

### 输出格式

共1行，为消除所有行星的最少次数。

## 输入输出样例

### 输入样例 #1

```cpp
3
1 2 3
2 3 1
1 3 2
```


### 输出样例 #1

```cpp
2
```


## 说明

1≤n≤50000

1≤x,y,z≤500
