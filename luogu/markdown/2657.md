# [SCOI2009]windy数

## 题目描述

windy定义了一种windy数。不含前导零且相邻两个数字之差至少为2的正整数被称为windy数。 windy想知道，

在A和B之间，包括A和B，总共有多少个windy数？

## 输入输出格式

### 输入格式

包含两个整数，A B。

### 输出格式

一个整数

## 输入输出样例

### 输入样例 #1

```cpp
1 10
```


### 输出样例 #1

```cpp
9
```


### 输入样例 #2

```cpp
25 50

```
### 输出样例 #2

```cpp
20

```
## 说明

100%的数据，满足 1 <= A <= B <= 2000000000 。
