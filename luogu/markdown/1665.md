# 正方形计数

## 题目描述

给定平面上N个点，你需要计算以其中4个点为顶点的正方形的个数。注意这里的正方形边不一定需要和坐标轴平行。

## 输入输出格式

### 输入格式

第一行一个数X，以下N个点的坐标。

【数据规模】

对于20%的数据，满足1≤N≤20;

对于100%的数据，满足1≤N≤500; -50≤X[i],Y[i]≤50，点不会重叠。

### 输出格式

一个数表示正方形的个数。

## 输入输出样例

### 输入样例 #1

```cpp
7
0 0
0 1
1 0
1 1
1 2
2 1
2 2
```


### 输出样例 #1

```cpp
3
```
