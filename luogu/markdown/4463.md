# [国家集训队] calc

## 题目描述

一个序列$a_1,\cdots,a_n$是合法的，当且仅当：

长度为给定的$n$。

$a_1,\cdots,a_n$都是$[1,A]$中的整数。

$a_1,\cdots,a_n$互不相等。

一个序列的值定义为它里面所有数的乘积，即$a_1\times a_2\times\cdots\times a_n$。

求所有不同合法序列的值的和。

两个序列不同当且仅当他们任意一位不一样。

输出答案对一个数$mod$取余的结果。

## 输入输出格式

### 输入格式

一行3个数，$A$，$n$，$mod$。意义为上面所说的。

### 输出格式

一行结果。

## 输入输出样例

### 输入样例 #1

```cpp
9 7 10007
```


### 输出样例 #1

```cpp
3611
```


## 说明

0:$A<=10,n

1..3:$A<=1000,n

4..9:$A<=10^9,n

10..19:$A<=10^9,n

全部:$modA>n+1$
