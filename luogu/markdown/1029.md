# 最大公约数和最小公倍数问题

## 题目描述

输入$2$个正整数$x_0,y_0(2 \le x_0<100000,2 \le y_0<=1000000)$,求出满足下列条件的$P,Q$的个数

条件:

1. $P,Q$是正整数

2. 要求$P,Q$以$x_0$为最大公约数,以$y_0$为最小公倍数.

试求:满足条件的所有可能的$2$个正整数的个数.

## 输入输出格式

### 输入格式

$2$个正整数$x_0,y_0$

### 输出格式

$1$个数，表示求出满足条件的$P,Q$的个数

## 输入输出样例

### 输入样例 #1

```cpp
3 60

```
### 输出样例 #1

```cpp
4

```
## 说明

$P,Q$有4种

1、3,60

2、15,12

3、12,15

4、60,3
