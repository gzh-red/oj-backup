# [SDOI2009]SuperGCD

## 题目描述

Sheng bill有着惊人的心算能力，甚至能用大脑计算出两个巨大的数的GCD（最大公约 数）！因此他经常和别人比赛计算GCD。有一天Sheng bill很嚣张地找到了你，并要求和你比 赛，但是输给Sheng bill岂不是很丢脸！所以你决定写一个程序来教训他。

## 输入输出格式

### 输入格式

共两行： 第一行：一个数A。 第二行：一个数B。

### 输出格式

一行，表示A和B的最大公约数。

## 输入输出样例

### 输入样例 #1

```cpp
12
54
```


### 输出样例 #1

```cpp
6
```


## 说明

对于20%的数据，0 < A , B ≤ 10 ^ 18。

对于100%的数据，0 < A , B ≤ 10 ^ 10000。
