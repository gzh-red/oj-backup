## 输入输出格式

### 输入格式

共两行。第一行为铁盘个数N（1<=N<=50)，第二行为N个不同的正整数，分别为从上到下的铁盘的半径R。（1<=R<=100）

### 输出格式

一个正整数，表示使铁盘从小到大有序需要的最少翻转次数。

## 输入输出样例

### 输入样例 #1

```cpp
5
2 4 3 5 1
```


### 输出样例 #1

```cpp
5
```
