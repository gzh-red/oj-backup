# 超级数

## 题目描述

设x的约数个数为f(x)，若f(n)>f(i)（0<i<n，i为整数），就把n称作supernumber。 给出m个数a1,a2,...,am，请分别输出不大于a1，不大于a2，...，不大于am的最大supernumber。 

## 输入输出格式

### 输入格式

第一行，一个数m

第2~m+1行，每行一个数，其中第i+1为ai

### 输出格式

共m行，每行一个数，其中第i行为不大于ai的最大supernumber

## 输入输出样例

### 输入样例 #1

```cpp
2
8
13
```


### 输出样例 #1

```cpp
6
12
```


## 说明

对于40%的数据，m=1

对于50%的数据，ai<=1000

对于70%的数据，ai<=2\*10^7

对于100%的数据，ai<=10^17，m<=100000
