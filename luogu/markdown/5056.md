# 【模板】插头dp

## 题目背景

ural 1519

陈丹琦《基于连通性状态压缩的动态规划问题》中的例题

## 题目描述

给出n*m的方格，有些格子不能铺线，其它格子必须铺，形成一个闭合回路。问有多少种铺法？

## 输入输出格式

### 输入格式

第1行，n,m(2<=n,m<=12)

从第2行到第n+1行，每行一段字符串(m个字符)，"*"表不能铺线，"."表必须铺

### 输出格式

输出一个整数，表示总方案数

## 输入输出样例

### 输入样例 #1

```cpp
4 4
**..
....
....
....
```


### 输出样例 #1

```cpp
2
```


### 输入样例 #2

```cpp
4 4
....
....
....
....
```


### 输出样例 #2

```cpp
6
```
