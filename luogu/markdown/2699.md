# 【数学1】小浩的幂次运算

## 题目描述

幂次运算让小浩觉得有些烦，最近他碰到了一些问题，他需要求出所有w^i使得l<=w^i<=r，于是他找到了你。

## 输入输出格式

### 输入格式

输入为一行，有三个数，分别是l,r,w。意义如题目所描述

### 输出格式

输出为一行，输出所有满足条件的数字，每两个数字中间有一个空格

如果没有满足条件的数字则输出-1

## 输入输出样例

### 输入样例 #1

```cpp
1 10 2
```


### 输出样例 #1

```cpp
1 2 4 8
```


### 输入样例 #2

```cpp
2 4 5
```


### 输出样例 #2

```cpp
-1
```


## 说明

对于100%的数据：1<=l<=r<=1\*10^18，1<=w<=10^9

保证数据的正确性
