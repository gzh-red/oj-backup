# 骑士的工作

## 题目背景

你作为一个村的村长，保卫村庄是理所当然的了.今天，村庄里来了一只恶龙，他有n个头，恶龙到处杀人放火。你着急了。不过天无绝人之路，现在来了一个骑士团。里面有m位成员（往下看）

## 题目描述

每个人都可以砍掉一个大小不超过（<=）z的头，要money个金币，求最小花费。

## 输入输出格式

### 输入格式

第一行两个整数 n m

下接n行，一个整数 表示n个头的大小。

下接m行，每个人可以砍的头大小或金币(金币==头的大小）。

### 输出格式

一个整数，最小花费。如果无解，输出“you died!”

## 输入输出样例

### 输入样例 #1

```cpp
2 3
5 
4
7 
8
4
```


### 输出样例 #1

```cpp
11
```


## 说明

1<=n,m
