# 公约数的和

## 题目描述

有一天，TIBBAR和LXL比赛谁先算出1~N这N个数中每任意两个不同的数的最大公约数的和。LXL还在敲一个复杂而冗长的程序，争取能在100s内出解。而TIBBAR则直接想1s秒过而获得完胜，请你帮他完成这个任务。

## 输入输出格式

### 输入格式

共一行，一个正整数N。

### 输出格式

共一行，一个数，为1~N这N个数中每任意两个不同的数的最大公约数的和。

## 输入输出样例

### 输入样例 #1

```cpp
10
```


### 输出样例 #1

```cpp
67
```


## 说明

对于40%的数据，2≤N≤2000.

对于100%的数据，2≤N≤2000000.
