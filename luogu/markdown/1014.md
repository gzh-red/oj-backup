# Cantor表

## 题目描述

现代数学的著名证明之一是Georg Cantor证明了有理数是可枚举的。他是用下面这一张表来证明这一命题的：

$1/1$ , $1/2$ , $1/3$ , $1/4$, $1/5$, …

$2/1$, $2/2$ , $2/3$, $2/4$, …

$3/1$ , $3/2$, $3/3$, …

$4/1$, $4/2$, …

$5/1$, …

…

我们以$Z$字形给上表的每一项编号。第一项是$1/1$，然后是$1/2$，$2/1$，$3/1$，$2/2$，…

## 输入输出格式

### 输入格式

整数$N$（$1≤N≤10000000$）

### 输出格式

表中的第$N$项

## 输入输出样例

### 输入样例 #1

```cpp
7

```
### 输出样例 #1

```cpp
1/4
```
