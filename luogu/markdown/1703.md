# 那个什么密码2

## 题目背景

https://www.luogu.org/problem/show?pid=1079

## 题目描述

与原题一模一样、具体不同请见输入格式

## 输入输出格式

### 输入格式

第一行输入密钥

第二行输入明文

第三行输入一个正整数M,代表操作个数.

每个操作将[a,b]的字符全部倒转,比如abc倒转为cba.

接下来M行,每行输入两个正整数a,b。

### 输出格式

输出一行,输出密文。

## 输入输出样例

### 输入样例 #1

```cpp
CompleteVictory
Wherethereisawillthereisaway
0
```


### 输出样例 #1

```cpp
Yvqgpxaimmklongnzfwpvxmniytm
```
