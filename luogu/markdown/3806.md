# 【模板】点分治1

## 题目背景

感谢hzwer的点分治互测。

## 题目描述

给定一棵有n个点的树

询问树上距离为k的点对是否存在。

## 输入输出格式

### 输入格式

n,m

接下来n-1条边a,b,c描述a到b有一条长度为c的路径

接下来m行每行询问一个K

### 输出格式

对于每个K每行输出一个答案，存在输出“AYE”,否则输出”NAY”(不包含引号)

## 输入输出样例

### 输入样例 #1

```cpp
2 1
1 2 2
2
```


### 输出样例 #1

```cpp
AYE
```


## 说明

对于30%的数据n<=100

对于60%的数据n<=1000,m<=50

对于100%的数据n<=10000,m<=100,c<=10000,K<=10000000
