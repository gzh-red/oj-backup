# [POI2013]GDZ-Where is the one?【交互题】

## 题目描述

交互题,已知一个排列,每次你可以询问1.某两个位置的数的差是否被某个数整除,2.第i个数是否大于第j个数。请用最少的2操作找出排列中1所在的的位置。操作1的次数是任意的,但是不能超时。n<=5\*10^5
