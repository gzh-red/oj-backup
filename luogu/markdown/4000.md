# 斐波那契数列

## 题目描述

大家都知道，斐波那契数列是满足如下性质的一个数列：

• f(1) = 1

• f(2) = 1

• f(n) = f(n-1) + f(n-2) (n ≥ 2 且 n 为整数)

请你求出 f(n) mod p 的值。

## 输入输出格式

### 输入格式

- 第 1 行：一个整数 n。

- 第 2 行：一个整数 p。

### 输出格式

- 第 1 行：f(n) mod p 的值。

## 输入输出样例

### 输入样例 #1

```cpp
5
1000000007
```


### 输出样例 #1

```cpp
5
```


### 输入样例 #2

```cpp
10
1000000007
```


### 输出样例 #2

```cpp
55
```
