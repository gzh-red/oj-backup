# 环状最大两段子段和

## 题目描述

给出一段环状序列，即认为$A_1$和$A_N$是相邻的，选出其中连续不重叠且非空的两段使得这两段和最大。

## 输入输出格式

### 输入格式

第一行是一个正整数$N(N\le 2\times 10^{5})$，表示了序列的长度。

第二行包含$N$个绝对值不大于$10000$的整数$A_i$，描述了这段序列，第一个数和第$N$个数是相邻的。

### 输出格式

一个整数，为最大的两段子段和是多少。

## 输入输出样例

### 输入样例 #1

```cpp
7
2 -4 3 -1 2 -4 3

```
### 输出样例 #1

```cpp
9
```


## 说明

【样例说明】

一段为$3$。
