# 数位平方和

## 题目描述

定义S(n)表示n的各个数位的k次方的和。定义H(n)=min{n,S(n),H(S(n))}。

求$\sum _{i=A} ^{B} {H(i)} \mod 10000007$

## 输入输出格式

### 输入格式

一行三个数K、A、B。

【数据规模】

对于20%的数据，满足1≤A、B≤50;

对于100%的数据，满足1≤A、B≤10^6，K≤6.

### 输出格式

B

一个数∑H(i) mod 10000007

i=A

## 输入输出样例

### 输入样例 #1

```cpp
2 1 5
```


### 输出样例 #1

```cpp
14
```
