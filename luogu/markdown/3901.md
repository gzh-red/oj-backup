# 数列找不同

## 题目描述

现有数列$A_1,A_2,\cdots,A_N$，Q 个询问$(L_i,R_i)$，$A_{Li} ,A_{Li+1},\cdots,A_{Ri}$ 是否互不相同

## 输入输出格式

### 输入格式

第1 行，2 个整数$N,Q$

第2 行，N 个整数$A_{Li} ,A_{Li+1},\cdots,A_{Ri}$

Q 行，每行2 个整数$L_i,R_i$

### 输出格式

对每个询问输出一行，“Yes” 或者“No”

## 输入输出样例

### 输入样例 #1

```cpp
4 2
1 2 3 2
1 3
2 4
```


### 输出样例 #1

```cpp
Yes
No
```


## 说明

• 对于50% 的数据，$N,Q \le 10^3$

• 对于100% 的数据，$1 \le N,Q \le 10^5, 1 \le A_i \le N, 1 \le L_i \le R_i \le N$
