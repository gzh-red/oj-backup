## 题目描述

有如下定义：若正整数序列$a$中存在连续若干个正整数的和为$m$的倍数，则这个正整数序列$a$被称为“司$m$序列”。

给定$n$和$m$，你需要知道长度为$n$的任意正整数序列$a$是否都是"司$m$序列"。

## 输入输出格式

### 输入格式

两个数，$n$和$m$。

### 输出格式

如果成立输出“YES”否则输出"NO"。

## 输入输出样例

### 输入样例 #1

```cpp
2 4
```


### 输出样例 #1

```cpp
NO
```


## 说明

样例解释：存在反例$1\ \ 2$。

$Subtask1(50pts)$：$1≤n,m≤5$

$Subtask2(50pts)$：$1≤n,m≤10^{18}$
