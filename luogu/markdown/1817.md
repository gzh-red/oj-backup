# 棋盘染色_NOI导刊2011提高（03）

## 题目描述

给定一个NxM的网格，每个格子可以染成黑色或者白色，要求所有黑色格子连通，所有白色格子连通，并且至少有一个黑色格子贴边，至少有一个白色格子贴边。问有多少种染色方法？ 

## 输入输出格式

### 输入格式

第一行有两个正整数N、M。

  

### 输出格式

只有一个正整数ANS。

## 输入输出样例

### 输入样例 #1

```cpp
TEST1：1 2   
TEST2：2 3
```


### 输出样例 #1

```cpp
TEST1：2   
TEST2：30
```


## 说明

对于100%的数据：N≤7，M≤8。
