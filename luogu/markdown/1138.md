# 第k小整数

## 题目描述

现有$n$个正整数，$n≤10000$，要求出这$n$个正整数中的第$k$个最小整数（相同大小的整数只计算一次），$k≤1000$，正整数均小于$30000$。

## 输入输出格式

### 输入格式

第一行为$n$和$k$; 第二行开始为$n$个正整数的值，整数间用空格隔开。

### 输出格式

第$k$个最小整数的值；若无解，则输出“NO RESULT”。

## 输入输出样例

### 输入样例 #1

```cpp
10 3
1 3 3 7 2 5 1 2 4 6

```
### 输出样例 #1

```cpp
3

```
## 说明

$n≤10000$
