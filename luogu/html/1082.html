同余方程

题目描述

求关于$ x$的同余方程 $ a x \equiv 1 \pmod {b}$ 的最小正整数解。

输入输出格式

输入格式


一行，包含两个正整数 $a,b$，用一个空格隔开。

输出格式


一个正整数 $x_0$，即最小正整数解。输入数据保证一定有解。

输入输出样例

输入样例 #1

3 10

输出样例 #1

7

说明

【数据范围】 对于 40%的数据，$2 ≤b≤ 1,000$； 对于 60%的数据，$2 ≤b≤ 50,000,000$； 对于 100%的数据，$2 ≤a, b≤ 2,000,000,000$。 NOIP 2012 提高组 第二天 第一题
