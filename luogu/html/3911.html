最小公倍数之和

题目描述

对于$A_1,A_2,\cdots,A_N$，求 $\sum_{i=1}^N\sum_{j=1}^N lcm(A_i,A_j)$ 的值。 $lcm(a,b)$ 表示a 和b 的最小公倍数

输入输出格式

输入格式


第1 行，1 个整数N。 第2 行，N 个整数$A_1,A_2,\cdots,A_N$。

输出格式


1 个整数，表示所求的值。

输入输出样例

输入样例 #1

2
2 3

输出样例 #1

17

说明

• 对于30% 的数据，$1 \le N \le 1000; 1 \le A_i \le 50000$； • 对于另外30% 的数据，$1 \le N \le 50000; 1 \le A_i \le 1000$； • 对于100% 的数据，$1 \le N \le 50000; 1 \le A_i \le 50000$。
