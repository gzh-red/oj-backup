yyy loves Maths II

题目背景

上次蒟蒻redbag可把yyy气坏了，yyy说他只是小学生，蒟蒻redbag这次不坑他了。

题目描述

redbag给了yyy很多个数，要yyy计算这些数的和。必须要快，redbag只给了yyy1秒的时间！！！

输入输出格式

输入格式


一行，很多个数

输出格式


一行,一个实数(四舍五入精确到5位小数)，表示这些数的和。

输入输出样例

输入样例 #1

1

输出样例 #1

1.00000

说明

[数据规模] 对于100%的数据,所有数<=233333333，保证在小数点后[b]最多[/b]有6位数,最多共有50000个数
