题目描述

![](https://cdn.luogu.com.cn/upload/pic/44005.png) 给你一个长为n的序列a，m次询问，每次查询一个区间的逆序对数

输入输出格式

输入格式


第一行两个数n，m 第二行n个数表示这个序列 之后m行，每行两个数表示查询的区间

输出格式


输出m行，每行一个数表示这次询问的答案

输入输出样例

输入样例 #1

4 1
1 4 2 3
2 4

输出样例 #1

2

说明

n,m <= 100000,0 <= ai <= 1000000000 我们已经有了低于n^1.5的算法 Source By nzhtl1477
