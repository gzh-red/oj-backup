修剪草坪

题目描述

在一年前赢得了小镇的最佳草坪比赛后，Farm John变得很懒，再也没有修剪过草坪。现在，新一轮的最佳草坪比赛又开始了，Farm John希望能够再次夺冠。 然而，Farm John的草坪非常脏乱，因此，Farm John只能够让他的奶牛来完成这项工作。Farm John有N(1 <= N <= 100,000)只排成一排的奶牛，编号为1...N。每只奶牛的效率是不同的，奶牛i的效率为E\_i(0 <= E\_i <= 1,000,000,000)。 靠近的奶牛们很熟悉，因此，如果Farm John安排超过K只连续的奶牛，那么，这些奶牛就会罢工去开派对:)。因此，现在Farm John需要你的帮助，计算FJ可以得到的最大效率，并且该方案中没有连续的超过K只奶牛。

输入输出格式

输入格式


第一行：空格隔开的两个整数 N 和 K 第二到 N+1 行：第 i+1 行有一个整数 E\_i

输出格式


第一行：一个值，表示 Farm John 可以得到的最大的效率值。

输入输出样例

输入样例 #1

5 2
1
2
3
4
5

输出样例 #1

12
