车的放置

题目描述

有下面这样的一个网格棋盘，a，b，c，d表示了对应边长度，也就是对应格子数。 ![](https://cdn.luogu.com.cn/upload/pic/142.png) 当a=b=c=d=2时，对应下面这样一个棋盘 ![](https://cdn.luogu.com.cn/upload/pic/143.png) 要在这个棋盘上放K个相互不攻击的车，也就是这K个车没有两个车在同一行，也没有两个车在同一列，问有多少种方案。同样只需要输出答案mod 100003后的结果。

输入输出格式

输入格式


输入文件place.in的第1行为有5个非负整数a, b, c, d和k。

输出格式


输出文件place.out包括1个正整数，为答案mod 100003后的结果。

输入输出样例

输入样例 #1

2 2 2 2 2

输出样例 #1

38

说明

【数据规模与约定】 对于部分数据，有b = 0； 对于部分数据，有a，b，c，d≤4。 对于100%的数据，a，b，c，d，k≤1000，且保证了至少有一种可行方案。
