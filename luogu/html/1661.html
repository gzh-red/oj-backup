扩散

题目描述

一个点每过一个单位时间就会向四个方向扩散一个距离，如图。 ![](https://cdn.luogu.com.cn/upload/pic/455.png) 两个点a、b连通，记作e(a,b),当且仅当a、b的扩散区域有公共部分。连通块的定义是块内的任意两个点u、v都必定存在路径e(u,a0),e(a0,a1),…,e(ak,v)。给定平面上的n给点，问最早什么时刻它们形成一个连通块。

输入输出格式

输入格式


第一行一个数n，以下n行，每行一个点坐标。 【数据规模】 对于20%的数据，满足1≤N≤5; 1≤X[i],Y[i]≤50; 对于100%的数据，满足1≤N≤50; 1≤X[i],Y[i]≤10^9。

输出格式


一个数，表示最早的时刻所有点形成连通块。

输入输出样例

输入样例 #1

2
0 0
5 5

输出样例 #1

5
