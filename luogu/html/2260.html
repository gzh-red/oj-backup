[清华集训2012]模积和

题目背景

数学题，无背景。

题目描述

求 $$\sum_{i=1}^{n} \sum_{j=1}^{m} (n \bmod i) \times (m \bmod j), i \neq j$$ mod 19940417 的值

输入输出格式

输入格式


两个整数n m

输出格式


答案 mod 19940417

输入输出样例

输入样例 #1

3 4

输出样例 #1

1

输入样例 #2

123456 654321

输出样例 #2

116430

说明

30%: n,m <= 1000 60%: n,m <= 10^6 100% n,m <= 10^9
