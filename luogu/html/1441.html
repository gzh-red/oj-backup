砝码称重

题目描述

现有n个砝码，重量分别为a1，a2，a3，……，an，在去掉m个砝码后，问最多能称量出多少不同的重量（不包括0）。 请注意，砝码只能放在其中一边。

输入输出格式

输入格式


输入文件weight.in的第1行为有两个整数n和m，用空格分隔 第2行有n个正整数a1，a2，a3，……，an，表示每个砝码的重量。

输出格式


输出文件weight.out仅包括1个整数，为最多能称量出的重量数量。

输入输出样例

输入样例 #1

3 1
1 2 2

输出样例 #1

3

说明

【样例说明】 在去掉一个重量为2的砝码后，能称量出1，2，3共3种重量。 【数据规模】 对于20%的数据，m=0； 对于50%的数据，m≤1； 对于50%的数据，n≤10； 对于100%的数据，n≤20，m≤4，m＜n，ai≤100。
