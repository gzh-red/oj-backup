[JSOI2015]最大公约数

题目描述

给定一个长度为$N$的正整数序列$A_i$。 对于其任意一个连续的子序列$A_l,A_{l+1},...,A_r$，我们定义其权值$W(L,R)$为其长度与序列中所有元素的最大公约数的乘积，即$W(L,R) = (R-L+1) × gcd (A_l,...,A_r)$。 `JYY` 希望找出权值最大的子序列。

输入输出格式

输入格式


输入一行包含一个正整数$N$。 接下来一行，包含$N$个正整数，表示序列$A_i$。

输出格式


输出文件包含一行一个正整数，表示权值最大的子序列的权值。

输入输出样例

输入样例 #1

5
30 60 20 20 20

输出样例 #1

80

说明

$1 \le A_i \le 10^{12}, 1 \le N \le 100000$
