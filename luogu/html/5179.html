Fraction

题目描述

给你四个正整数 $a,\,b,\,c,\,d$ ，求一个最简分数 $\frac{p}{q}$ 满足 $\frac{a}{b} < \frac{p}{q} < \frac{c}{d}$。 若有多组解，输出 $q$ 最小的一组，若仍有多组解，输出 $p$ 最小的一组。

输入输出格式

输入格式


本题有多组数据，有若干行，每行四个正整数，为每组数据的 $a,\,b,\,c,\,d$。

输出格式


每组数据各输出一行，如果答案是 $\frac{p}{q}$，则以 `p/q` 的格式输出。

输入输出样例

输入样例 #1

1 3 1 2
2 1 3 1
2 1 4 1
1000 1001 1001 1002

输出样例 #1

2/5
5/2
3/1
2001/2003

说明

|||| |:-:|:-:|:-:| |测试点编号|数据组数|$a,\,b,\,c,\,d$| |$1 \sim 3$|$\leqslant 100$|$\leqslant 100$| |$4 \sim 10$|$\leqslant 500$|$\leqslant 10^9$| 数据保证至少存在一个最简分数符合条件。
