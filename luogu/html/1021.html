邮票面值设计

题目描述

给定一个信封，最多只允许粘贴$N$张邮票，计算在给定$K$（$N+K≤15$）种邮票的情况下（假定所有的邮票数量都足够），如何设计邮票的面值，能得到最大值$MAX$，使在$1$至$MAX$之间的每一个邮资值都能得到。 例如，$N=3$，$K=2$，如果面值分别为$1$分、$4$分，则在$1$分～$6$分之间的每一个邮资值都能得到（当然还有$8$分、$9$分和$12$分）；如果面值分别为$1$分、$3$分，则在$1$分～$7$分之间的每一个邮资值都能得到。可以验证当$N=3$，$K=2$时，$7$分就是可以得到的连续的邮资最大值，所以$MAX=7$，面值分别为$1$分、$3$分。

输入输出格式

输入格式


$2$个整数，代表$N$，$K$。

输出格式


$2$行。第一行若干个数字，表示选择的面值，从小到大排序。 第二行，输出“$MAX=S$”，$S$表示最大的面值。

输入输出样例

输入样例 #1

3 2

输出样例 #1

1 3
MAX=7
