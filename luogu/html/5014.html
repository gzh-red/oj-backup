题目背景

这个三角图真好看。。 ![](https://i.loli.net/2018/10/30/5bd7cc3fa1bfb.png) 这个是${\rm 4}$阶三角图。。

题目描述

现在我们定义一个三角图是像上面一样的图。。 请求出一个无限大的三角图从$1$号点走到$u$号点的方案数。 有$T$组询问。

输入输出格式

输入格式


第一行一个正整数$T$ 第二行$T$个正整数$u_i$。

输出格式


$T$行，共$T$个正整数，表示答案模$998244353$的结果。

输入输出样例

输入样例 #1

3
1 3 6

输出样例 #1

1
2
6

说明

${\rm Subtask 1(10\ pts)}$:$1 \leq T \leq 100, \qquad 1 \leq u_i \leq 55$ ${\rm Subtask 2(20\ pts)}$:$1 \leq T \leq 100, \qquad 1 \leq u_i \leq 12502500$ ${\rm Subtask 3(30\ pts)}$:$1 \leq T \leq 100, \qquad 1 \leq u_i \leq 500000500000 \qquad u_i=\frac{x \times (x + 1)}{2}$ ${\rm Subtask 4(40\ pts)}$:$1 \leq T \leq 100, \qquad 1 \leq u_i \leq 500000500000$
