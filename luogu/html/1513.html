绕钉子的长绳子

题目背景

平面上有N个圆柱形的大钉子，半径都为R,所有钉子组成一个凸多边形。 现在你要用一条绳子把这些钉子围起来，绳子直径忽略不计。

题目描述

求出绳子的长度

输入输出格式

输入格式


第1行两个数：整数N(1<=N<=100)和实数R。 接下来N行按逆时针顺序给出N个钉子中心的坐标 坐标的绝对值不超过100。

输出格式


一个数，绳子的长度，精确到小数点后2位。

输入输出样例

输入样例 #1

4 1
0.0 0.0
2.0 0.0
2.0 2.0
0.0 2.0

输出样例 #1

14.28
