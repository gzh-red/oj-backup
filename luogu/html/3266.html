题目描述

说起来，毕业之后 B 君也就见过 R 君两面而已。 R 君有一个 $n \times m$ 的数组 $x_{i,j}(1 \le i \le n; 1 \le j \le m)$。 对于 $1 \le i \le n; 1 \le j \le m$，满足$0 \le x_i; j \le m$。求 可能的数组$x_{i,j}$ 的解数。 B 君觉得限制太宽松，还要求对于 $1 \le i \le n; 1 \le j

输入输出格式

输入格式


一行两个整数表示 n, m，含义如题目中所述。

输出格式


一行一个数表示同时满足 B 君和 R 君的条件{xi,j} 的解数，模 1000000007 的结果。

输入输出样例

输入样例 #1

3 3

输出样例 #1

40

说明

对于 100% 的数据，1 <= m;n <= 10^6
