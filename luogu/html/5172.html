Sum

题目描述

给定正整数 $n,\,r$ ，求： $$\sum_{d=1}^{n}(-1)^{\lfloor d\sqrt{r} \rfloor}$$

输入输出格式

输入格式


第一行一个数 $t$，表示有 $t$ 组测试数据。 接下来 $t$ 行，每行两个正整数 $n,\,r$ 。

输出格式


输出 $t$ 行，每行一个整数表示答案。

输入输出样例

输入样例 #1

3
3 5
3 6
3 7

输出样例 #1

3
1
-1

说明
