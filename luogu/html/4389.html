付公主的背包

题目背景

付公主有一个可爱的背包qwq

题目描述

这个背包最多可以装$10^5$大小的东西 付公主有n种商品，她要准备出摊了 每种商品体积为Vi，都有$10^5$件 给定m，对于$s\in [1,m]$，请你回答用这些商品恰好装s体积的方案数

输入输出格式

输入格式


第一行n,m 第二行V1~Vn

输出格式


m行，第i行代表s=i时方案数，对998244353取模

输入输出样例

输入样例 #1

2 4
1 2

输出样例 #1

1
2
2
3

说明

对于30%的数据，n<=3000,m<=3000 对于60%的数据，纯随机生成 对于100%的数据， n<=100000,m<=100000 对于100%的数据，Vi<=m
