【模板】多项式反三角函数

题目描述

给定一个$n-1$次多项式$A(x)$，求一个$\bmod{\:x^n}$下的多项式$F(x)$，满足$F(x)\equiv\text{asin}\:A(x)$或$F(x)\equiv\text{atan}\:A(x)$。 所有运算在模$998244353$意义下进行。

输入输出格式

输入格式


第一行**两个**整数$n,type$，若$type=0$代表求$\text{asin}$，若$type=1$代表求$\text{atan}$； 第二行$n$个整数，依次表示多项式的系数$a_0,a_1,\cdots,a_{n-1}$。 保证$a_0=0$

输出格式


输出一行$n$个整数，表示答案多项式中的系数$f_0,f_1,\cdots,f_{n-1}$。

输入输出样例

输入样例 #1

8 0
0 4 2 6 1 5 3 7

输出样例 #1

0 4 2 665496252 17 399297879 332748370 570426983

输入样例 #2

8 1
0 4 2 6 1 5 3 7

输出样例 #2

0 4 2 665496220 998244322 399297839 332748518 570424795

说明

对于$100\%$的数据：$n\leq 10^5$，$a_i\in[0,998244352]\cap\mathbb{Z}$ 前5个点$type=0$，后5个点$type=1$
