# Kanban Numbers

## 题意翻译

输入

包含一个整数 a (1<=a

输出

"YES"或者"NO"

## 题目描述

## 输入输出格式

### 输入格式

The input contains a single integer $ a $ ( $ 1 \le a \le 99 $ ).

### 输出格式

Output "YES" or "NO".

## 输入输出样例

### 输入样例 #1

```cpp
5

```
### 输出样例 #1

```cpp
YES

```

### 输出样例 #2
