# Quirky Quantifiers

## 题意翻译

CF愚人节题目，自己想。

输入一个整数a( 10<=a

输出0或1.

## 题目描述

## 输入输出格式

### 输入格式

The input contains a single integer $ a $ ( $ 10<=a<=999 $ ).

### 输出格式

Output 0 or 1.

## 输入输出样例

### 输入样例 #1

```cpp
13

```
### 输出样例 #1

```cpp
1

```
### 输入样例 #2

```cpp
927

```
### 输出样例 #2

```cpp
1

```
### 输入样例 #3

```cpp
48

```
### 输出样例 #3

```cpp
0

```
