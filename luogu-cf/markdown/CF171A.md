# Mysterious numbers - 1

## 题意翻译

输入格式：

输入包括两个整数a1,a2,(0<=ai

输出格式：

输出一个整数

## 题目描述

## 输入输出格式

### 输入格式

The input contains two integers $ a_{1},a_{2} $ ( $ 0<=a_{i}<=10^{9} $ ), separated by a single space.

### 输出格式

Output a single integer.

### 输出样例 #1

```cpp
44

```

