# Word Capitalization

## 题意翻译

题目描述

大写是把第一个字母改成一个大写字母。你的任务是把给定的单词大写。

注意，在大写期间，除了第一个字母以外，所有字母保持不变。

输入输出格式

输入格式：

仅一行，一个单词。这个词由小写字母和大写英文字母组成。这个词的长度不会超过1000。

输出格式：

在大写之后输出给定的单词。

## 题目描述

Capitalization is writing a word with its first letter as a capital letter. Your task is to capitalize the given word.

Note, that during capitalization all the letters except the first one remains unchanged.

## 输入输出格式

### 输入格式

A single line contains a non-empty word. This word consists of lowercase and uppercase English letters. The length of the word will not exceed $ 10^{3} $ .

### 输出格式

Output the given word after capitalization.

## 输入输出样例

### 输入样例 #1

```cpp
ApPLe

```
### 输出样例 #1

```cpp
ApPLe

```
### 输入样例 #2

```cpp
konjac

```
### 输出样例 #2

```cpp
Konjac

```
