# Asterisks【卡语言已隐藏】

## 题目描述

You are given a number $ n $ . Print $ n $ lines, $ i $ -th line should consist of $ i $ characters "\*". Lines' indices are 1-based.

## 输入输出格式

### 输入格式

The only line of input contains an integer $ n $ ( $ 1<=n<=50 $ ).

### 输出格式

Output the described pattern.

## 输入输出样例

### 输入样例 #1

```cpp
3

```
### 输出样例 #1

```cpp
*
**
***

```
### 输入样例 #2

```cpp
6

```
### 输出样例 #2

```cpp
*
**
***
****
*****
******

```
