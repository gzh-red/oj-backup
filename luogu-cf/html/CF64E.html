Prime Segment【卡语言已隐藏】

题目描述

Positive integer number $ x $ is called prime, if it has exactly two positive integer divisors. For example, 2, 3, 17, 97 are primes, but 1, 10, 120 are not. You are given an integer number $ n $ , find the shortest segment \[ $ a,b $ \], which contains $ n $ (i.e. $ a<=n<=b $ ) and $ a,b $ are primes.

输入输出格式

输入格式


The only given line contains an integer number $ n $ ( $ 2<=n<=10000 $ ).

输出格式


Print the space separated pair of the required numbers $ a $ , $ b $ .

输入输出样例

输入样例 #1

10

输出样例 #1

7 11

输入样例 #2

97

输出样例 #2

97 97
