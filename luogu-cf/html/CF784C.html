输入输出格式

输入格式


The first line of input data contains a single integer $ n $ ( $ 1<=n<=10 $ ). The second line of input data contains $ n $ space-separated integers $ a_{i} $ ( $ 1<=a_{i}<=11 $ ).

输出格式


Output a single integer.
