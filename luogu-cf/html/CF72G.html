Fibonacci army

题意翻译

输出斐波那契数列的第n项，斐波那契数列公式：f[0]:=1;f[1]:=1;f[n]:=f[n-1]+fn-2

题目描述

King Cambyses loves Fibonacci numbers. He has several armies. Today he wants to make a new army for himself and he wants the number of men in this army to be the $ n $ -th Fibonacci number. Given $ n $ you should find $ n $ -th Fibonacci number. The set of Fibonacci numbers start with $ f_{0}=f_{1}=1 $ and for each $ i>=2 $ , $ f_{i}=f_{i-1}+f_{i-2} $ .

输入输出格式

输入格式


Input contains a single integer $ n $ ( $ 1<=n<=20 $ ).

输出格式


Write a single integer. The $ n $ -th Fibonacci number.

输入输出样例

输入样例 #1

2

输出样例 #1

2

输入样例 #2

1

输出样例 #2

1
