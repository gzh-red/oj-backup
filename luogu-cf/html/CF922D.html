Robot Vacuum Cleaner

题意翻译

有n个由s,h 组成的串 这些串可以按照任意顺序连接 最大化sh子序列（不用连续）的数量

题目描述

While moving, the robot generates a string $ t $ consisting of letters 's' and 'h', that produces a lot of noise. 输入输出格式

输入格式


The first line contains a single integer $ n $ ( $ 1<=n<=10^{5} $ ) — the number of strings in robot's memory. Next $ n $ lines contain the strings $ t_{1},t_{2},...,t_{n} $ , one per line. It is guaranteed that the strings are non-empty, contain only English letters 's' and 'h' and their total length does not exceed $ 10^{5} $ .

输出格式


Print a single integer — the maxumum possible noise Imp can achieve by changing the order of the strings.

输出样例 #1

输出样例 #2

1

说明

The optimal concatenation in the first sample is $ ssshhshhhs $ .
