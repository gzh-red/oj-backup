Calculating Function

题意翻译

对于正整数n，我们定义函数f： f（n）＝-1+2-3＋…+(-1)^n *n 您的任务是计算f（n）。 输入输出格式 输入： 一行，为正整数n（1

题目描述

For a positive integer $ n $ let's define a function $ f $ : $ f(n)=-1+2-3+..+(-1)^{n}n $ Your task is to calculate $ f(n) $ for a given integer $ n $ .

输入输出格式

输入格式


The single line contains the positive integer $ n $ ( $ 1<=n<=10^{15} $ ).

输出格式


Print $ f(n) $ in a single line.

输入输出样例

输入样例 #1

4

输出样例 #1

2

输入样例 #2

5

输出样例 #2

-3

说明

$ f(4)=-1+2-3+4=2 $ $ f(5)=-1+2-3+4-5=-3 $
