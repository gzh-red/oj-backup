The first input line contains a single integer $ n $ . The second line contains $ n $ integers $ a_{i} $ ( $ 0<=a_{i}<=10^{4} $ ), separated by single spaces. The input limitations for getting 20 points are: - $ 1<=n<=300 $ The input limitations for getting 50 points are: - $ 1<=n<=2000 $ The input limitations for getting 100 points are: - $ 1<=n<=10^{5} $

输出格式


Print exactly $ n-1 $ lines: the $ k $ -th output line must contain the minimum number of moves needed to make the first $ k $ elements of the original sequence $ a_{i} $ equal to zero.
