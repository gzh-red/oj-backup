Magic Numbers

题意翻译

魔数是由1，14，144组成的，每个数字可以重复使用，例如：14144，141414和1411就是魔数，而1444，514 和414就不是魔数。判断一个数是不是魔数。

题目描述

A magic number is a number formed by concatenation of numbers $ 1 $ , $ 14 $ and $ 144 $ . We can use each of these numbers any number of times. Therefore $ 14144 $ , $ 141414 $ and $ 1411 $ are magic numbers but $ 1444 $ , $ 514 $ and $ 414 $ are not. You're given a number. Determine if it is a magic number or not.

输入输出格式

输入格式


The first line of input contains an integer $ n $ , $ (1<=n<=10^{9}) $ . This number doesn't contain leading zeros.

输出格式


Print "YES" if $ n $ is a magic number or print "NO" if it's not.

输入输出样例

输入样例 #1

114114

输出样例 #1

YES

输入样例 #2

1111

输出样例 #2

YES

输入样例 #3

441231

输出样例 #3

NO
