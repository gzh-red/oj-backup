Mysterious strings

题意翻译

注意:本题为CodeForces 2013年愚人节的题目... 输入一个数，输出这个数字对应的一任美国总统的英文名字(到里根总统截止)，连任仅算一任。 提示:历任美国总统 string arrPresidents[40] = { "Washington", "Adams", "Jefferson", "Madison", "Monroe", "Adams", "Jackson", "Van Buren", "Harrison", "Tyler", "Polk", "Taylor", "Fillmore", "Pierce", "Buchanan", "Lincoln", "Johnson", "Grant", "Hayes", "Garfield", "Arthur", "Cleveland", "Harrison", "Cleveland", "McKinley", "Roosevelt", "Taft", "Wilson", "Harding", "Coolidge", "Hoover", "Roosevelt", "Truman", "Eisenhower", "Kennedy", "Johnson", "Nixon", "Ford", "Carter", "Reagan" };

题目描述

输入输出格式

输入格式


The input contains a single integer $ a $ ( $ 1<=a<=40 $ ).

输出格式


Output a single string.

输入输出样例

输入样例 #1

2

输出样例 #1

Adams

输入样例 #2

8

输出样例 #2

Van Buren

输入样例 #3

29

输出样例 #3

Harding
